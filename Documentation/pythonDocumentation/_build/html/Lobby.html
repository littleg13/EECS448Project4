Handles the collision of bullets

Checks if the bullet will collide with either a wall or a player, and generates updated game information

Parameters:
  • userID (string) – Unique key generated for the player that fired
  • player (object) – The entire player object of the player that fired
  • power (int) – Power of the projectile fired(set to zero in first release)
  • spin (int) – Spin of the projectile fired(set to zero in first release)
Returns:

first element is the ID of the player that was collided with,

or false if there was no collision. Second element is the distance from the player who shot to whatever it collided with
Return type:

array [string, float]
